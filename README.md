# 宏观风险判断 · 公开研究日志

一个人维护的全球宏观风险判断清单,以及对这些判断做对抗式审计后得到的「可证伪预测」与命中记录。

🌐 **在线展示**:https://macro-public.pages.dev/ (Cloudflare Pages,内容随本仓库每次更新)

## 内容

| 文件 | 是什么 |
|---|---|
| `index.html` | 展示网页(抓取下面几个文件渲染),部署在 Cloudflare Pages |
| `snapshot.md` | 当前宏观风险判断清单(滚动更新) |
| `log.md` | 判断清单的历史演变(append-only,每期=日期→相对上期净变化→指向快照) |
| `scorecard.json` | 可证伪预测 + 命中记录(校准记分卡,仅含标记公开的条目) |
| `methodology.md` | 这套「判断 → 审计判断」的方法是怎么运作的 |
| `transmission-map.md` | 风险之间的传导关系(已泛化) |

## 这个项目在做什么

判断由人持有并维护;一组目标函数不同的 AI 角色(红队、基率/外部视角、二阶传导、已定价、逆向)对清单做对抗式审计,逼出每条判断的证伪点;现实揭晓后用结果验证,累积成命中记录。详见 `methodology.md`。
