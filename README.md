# 宏观风险判断 · 公开研究日志

一个个人维护的全球宏观风险判断清单,以及对这些判断做对抗式审计后得到的"可证伪预测"与命中记录。

这里只放可公开的内容:当前判断清单、历史演变、判断的方法论、以及一张随时间累积的校准记分卡。具体的投资持仓与个人化推理不在此公开。

## 内容

- `snapshot.md` — 当前宏观风险判断清单(滚动更新)
- `log.md` — 判断清单的历史演变(append-only)
- `transmission-map.md` — 风险之间的传导关系(已泛化)
- `scorecard.json` — 可证伪预测 + 命中记录(校准记分卡)
- `methodology.md` — 这套判断与审计方法是怎么运作的

## 关于这个项目

判断由人持有并维护;一组目标函数不同的 AI 角色(红队、基率/外部视角、二阶传导、已定价、逆向)对清单做对抗式审计,逼出每条判断的证伪点;现实揭晓后用结果验证,累积成命中记录。详见 `methodology.md`。
