# 风险传导图(公开 · 已泛化)

> 记录宏观风险之间的传导关系。已去除任何与具体持仓相关的内容,只保留一般性的传导逻辑。

## 关键边

- **E18**:AI 资本开支 → 美国增长的重要支撑 → 若 capex/信用链断裂 → 增长→衰退→周期 → 打**工业/周期类资产**。这是宏观风险与周期类股票风险的主要交汇点,且不挑利率环境(regime-invariant)。
- **E19**:AI 内存涨价(HBM 挤占 DRAM/NAND)→ 汽车/电子等 BOM 成本上行(弱-中边际压力)。
- **E20**:AI 电力竞争 → 欧洲能源成本(弱-中边际压力)。

## 一个被低估的传导格

油价下跌 + AI 过剩 + 央行仍按兵(就业未断)= **无缓冲的增长冲击**:增长侧受伤,却拿不到降息缓冲,因为央行对就业是后视的。监测乘积:

> 油价符号 ×(当世代 GPU 租金 / HBM 现货 ASP 符号)= 尾部坐标。
